# Examples

This directory can contain example files for testing and reference.

## Example Files (not included in repo)

Due to sensitive data, actual timesheet and MSR files are not stored in the repository.

For testing, place your files here:
- `timesheet_example.csv` - Sample timesheet export from TSheets
- `to1_example.xlsx` - Sample TO1 MSR
- `to4_example.xlsx` - Sample TO4 MSR
- `to6_example.xlsx` - Sample TO6 MSR

These files are gitignored and won't be committed.
